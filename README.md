# Personal Portfolio Website 🚀

A responsive, multi-section personal portfolio website built with HTML, CSS, and JavaScript — designed to showcase projects, skills, and contact information for data science and AI roles.

 
📌 About This Project

Problem: Resumes alone can't fully communicate a developer's skills, personality, or project depth. Recruiters and hiring managers increasingly look for an online presence that *shows* — not just tells — what a candidate can do.

Solution: A clean, fully responsive portfolio website that loads fast, works on all screen sizes, and presents projects, technical skills, and contact details in a structured, professional layout — built from scratch with zero frameworks or dependencies.

🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Structure | HTML5 (semantic markup) |
| Styling | CSS3 — Flexbox, Grid, responsive breakpoints |
| Behaviour | Vanilla JavaScript (no frameworks) |
| Deployment | Vercel |


✨ Features

- Responsive navbar with hamburger menu toggle for all screen sizes
- Hero section with name, tagline, and CTA button
- About section— skills in Python, SQL, Excel, Power BI, and ML
- Projects section showcasing:
  - 🔍 Forensic Accounting AI— ML-based financial statement manipulation detector using Python, Pandas, SQL, Power BI, NumPy, and Matplotlib. Trained on labeled datasets of genuine and manipulated financial statements. Built interactive Power BI dashboards to visualize anomalies and fraud patterns.
  - ♻️ AI & IoT Waste Management System— Smart waste monitoring using IoT sensors and AI analytics to enable real-time tracking, optimized collection routes, and improved recycling outcomes.
- Certifications section— Data Analytics & Python (Coursera)
- Contact formwith client-side validation
